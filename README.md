## Installation

```bash
$ npm install
```

## Running the App

```bash
# Development
$ npm run start

# Watch Mode
$ npm run start:dev

# Production Mode
$ npm run start:prod
```

## Test

```bash
# Unit Tests
$ npm run test

# e2e Tests
$ npm run test:e2e

# Test Coverage
$ npm run test:cov
```
